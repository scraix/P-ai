> 把 LLM 變成真正會幹活的桌面 AI 指揮部。  
> 它不只是聊天視窗，而是一個能管理自己、管理手下、管理技能、管理工作空間，並長期持續執行任務的桌面 AI 作業系統。

Easy Call AI 的目標，從來都不是「再做一個 AI 聊天網頁」。

它要做的是另一件更有野心的事：

- 讓 AI 可以被一鍵召喚到桌面邊緣
- 讓 AI 擁有穩定人格、穩定部門、穩定技能、穩定 MCP、穩定工作空間
- 讓 AI 不只回答問題，還能接任務、追任務、委派任務、長期協作
- 讓 AI 的記憶不再昂貴、短命、脆弱，而是可持續、可追蹤、可歸檔

## 核心定位

Easy Call AI 更接近下面這幾樣東西的結合：

- 桌面 AI 浮窗
- 長期運行的任務系統
- 可委派的部門 / 工具人體系
- Skill + MCP 工作區
- 低成本持續記憶系統
- 可自動歸檔與整理上下文的會話系統

一句話說完：

> 這是一個可召喚、可委派、可執行、可積累的桌面 AI 工作中樞。

## 核心能力

- 全域熱鍵召喚桌面 AI
- 多模型、多供應商統一運行
- 任務系統與任務看板
- 部門委派 / 工具人工作流
- Skill 工作區與預設技能
- MCP 接入與能力治理
- 私有部門、私有人格、私有工作區
- 主動回憶、私有記憶、低成本長期記憶
- 自動歸檔與上下文整理

## 適合誰

- 想把 LLM 真正融入桌面工作流的人
- 不滿足於「問一句答一句」的人
- 需要長期任務推進的人
- 希望 AI 具備組織能力、委派能力、記憶能力的人

## 快速開始

1. 啟動應用
2. 在系統托盤中打開設定視窗
3. 新增並保存 LLM API 設定
4. 選擇主助理部門的模型與委任人格
5. 設定呼喚熱鍵
6. 用熱鍵呼出對話視窗開始協作

## 隱私與資料

- API Key 預設保存在本機
- 對話、歸檔、任務、記憶、工作區資料預設保存在本地
- 你可以自行管理、匯出、清理這些資料
