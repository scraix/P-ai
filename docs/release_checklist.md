# 发布验收清单

> 更新于 2026-03-13

## 发布前

1. `pnpm install`
2. `pnpm smoke`
3. `git status` 无未提交改动

## 功能验收

### API 配置

- 独立配置对话 / 音转文 / 图转文 API
- 重启应用后配置正确

### 文本对话

- 发送纯文本，流式输出正常
- 停止按钮可中断流式

### 图片 / 音频回退

- 对话 API 关闭图片 → 配置 Vision API → 粘贴图片发送 → 正常回复
- 对话 API 关闭音频 → 配置 STT API → 录音发送 → 转写并入消息
- 同一张图二次发送 → 缓存命中

### 文件附件

- 拖拽文件到输入框 → 文件复制到 workspace downloads
- 发送后气泡显示文件名图标
- 附件信息以 JSON 存入 provider_meta

### 终端工具

- LLM 调用终端执行命令
- 命令输出正确返回
- 超时命令正确终止

### MCP 服务器

- 添加 / 启用 / 禁用 MCP 服务器
- LLM 调用 MCP 工具正常执行

### 任务调度

- 创建定时任务
- 任务按时触发消息入队

### 部门与委派

- 创建部门，绑定 agent 和 API
- 委派子会话正常创建和回报

### 记忆系统

- 记忆写入和召回正常
- 私有记忆隔离正确

### 对话归档

- 闲置对话自动归档
- 归档对话可查看

## 窗口与交互

- 全局快捷键打开/隐藏对话窗口
- 托盘菜单可打开配置/对话/归档窗口
- 对话窗口可拖动、可关闭、置顶可切换

## 回归兜底

1. `cargo test --no-run` 通过
2. `pnpm -s exec tsc --noEmit` 通过
