# 20260319 后台语音关键词截图归档

## 结论

本功能已完成并经用户反馈“效果完美”。

## 实现范围（按最终落地）

1. 对话设置新增“后台语音截图关键词”输入框
- 关键词输入支持中英文混输。
- 关键词保存改为手动：输入框右侧单独“保存”按钮。
- 保存时自动将全角逗号 `，` 归一化为半角逗号 `,`。

2. 对话设置新增“后台语音截图范围”切换
- 提供 `全屏` / `前台窗口` 两个按钮。
- 该项保留按钮式即时生效（点击即保存）。

3. 后台语音转写后自动判定截图
- 仅在后台语音激活路径执行关键词匹配。
- 匹配逻辑采用“去全部空白 + 全部小写”后再做包含匹配。
- 命中时按范围执行截图并注入附件队列。
- 失败不阻断语音主流程，并输出中文日志（命中/模式/结果/耗时/原因）。

4. 配置持久化与兼容
- 新增字段写入 app_data/runtime_state，并接入 load/save_chat_settings。
- 老数据兼容：当关键词为空时自动补默认关键词。

5. 默认值调整（按最终产品决策）
- 默认关键词：`看看,这个,屏幕上,see,look,watch`
- 默认范围：`前台窗口 (focused_window)`
